Tabla Comparativa de Eventos en JavaScript

Nombre del Evento Descripción Ejemplo de Uso
Al hacer clic (onclick) Se ejecuta cuando el usuario hace clic sobre un elemento. <button onclick="alert('¡Hola!')">Haz clic aquí</button>
Al pasar el mouse (onmouseover) Ocurre cuando el puntero del mouse pasa sobre un elemento. <div onmouseover="this.style.color='red'">Pasa el mouse por aquí</div>
Al presionar una tecla (onkeydown) Se activa cuando el usuario presiona una tecla del teclado. <input onkeydown="alert('Presionaste una tecla')" placeholder="Escribe algo...">
Al cambiar el texto (onchange) Se ejecuta cuando cambia el valor de un campo de entrada. <input type="text" onchange="alert('Texto cambiado')" placeholder="Escribe y cambia...">
Al cargar la página (onload) Se activa cuando la página web termina de cargarse. <body onload="alert('Página cargada correctamente')"></body>
